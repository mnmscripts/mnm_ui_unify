
# MnM UI Unify

Sync your UI files easy from one character to others. Maintains backups of all of your settings to %appdata%/../LocalLow/Niche Worlds Cult.

## Windows
Install a current version of python, example (for Intel/AMD): https://www.python.org/ftp/python/3.14.0/python-3.14.0-amd64.exe

Download the .py script above, double click to run.
